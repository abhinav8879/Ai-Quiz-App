# 🧠 QuizAI Platform

An advanced, interactive AI-driven learning platform designed to test knowledge across various domains through adaptive quizzes. Built with a focus on modern UI/UX, gamification, and real-time performance analytics.

### ✨ Key Features
* **Gamified Dashboard:** Track XP, coins, and global ranking.
* **Live Analytics:** Real-time performance tracking using Chart.js.
* **Adaptive UI:** Premium glassmorphism design with Dark/Light mode toggle.
* **Teacher Panel:** Admin interface for quiz management and data export.

### 💻 Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Variables, Animations), Vanilla JavaScript
* **Libraries:** Chart.js (Data Visualization), FontAwesome (Icons)
* **Deployment:** GitHub Pages
