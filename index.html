About This Project

The AI-Based Online Quiz Application with Adaptive Difficulty Level is an advanced educational platform developed as a Major Project for the BCA program (2023–2026) at Chandigarh University.

Designed by ABHINAV KUMAR, this system replaces static assessments with an intelligent engine. It dynamically adjusts the difficulty level of questions based on real-time user performance, ensuring a highly personalized and gamified learning experience. The integration of live analytics helps track precise knowledge gaps instantly.
